# The Hub POS

A Point of Sale application for The Hub Group.

## Installation

To install the dependencies, run:

```bash
npm i
```

## Usage

To start the development server, run:

```bash
npm run dev
```

To build the project for production, run:

```bash
npm run build
```

To preview the production build, run:

```bash
npm run preview
```
